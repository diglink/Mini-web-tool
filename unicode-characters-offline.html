You don't need to use images, SVGs or Fonts for icons !
Here are short-listed Unicode Characters which can be used as icons for designing websites.
These characters do not need any external files! Just Copy & paste to use it & improve page loading time.
Make sure you edit the files in UTF-8 mode in editor & add <meta charset="utf-8" /> in <head> of website.
